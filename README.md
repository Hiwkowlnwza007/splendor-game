# 💎 Splendor (Console Version)
**Final Project FRA143**

## 📌 Overview
* **ชื่อเกม:** Splendor (สเปลนเดอร์ - เกมค้าเพชร)
* **แนวเกม:** Strategy Board Game, Engine Building, Resource Management
* **จำนวนผู้เล่น:** 1 - 4 คน (เล่นบนเครื่องเดียวกัน - Local Multiplayer)
* **Platform:** Console / Terminal (รันได้ทั้งบน Windows, macOS และ Linux)
* **Tech Stack / Library:** * ภาษาที่ใช้พัฒนา: **C++** (Object-Oriented Programming)
  * Library: `<iostream>`, `<vector>`, `<string>`, `<fstream>`, `<sstream>`, `<algorithm>`, `<random>` (ใช้ Standard Library พื้นฐานทั้งหมด ไม่มี External Library)

## 📜 How to Play

### 📖 Rules (กติกาของเกม)
ผู้เล่นจะได้รับบทเป็นพ่อค้าอัญมณีในยุคเรอเนสซองส์ แข่งขันกันสะสมเพชร ซื้อการ์ดพัฒนา และดึงดูดชนชั้นสูง (ขุนนาง) เพื่อสะสม **"คะแนนชื่อเสียง" (Prestige Points)** ให้ครบ 15 คะแนนเป็นคนแรก

**องค์ประกอบของเกม:**
1. **เหรียญอัญมณี (Gems):** มี 5 สี (ขาว, ฟ้า, เขียว, แดง, ดำ) และเหรียญทอง (ใช้แทนสีอะไรก็ได้)
2. **การ์ดพัฒนา (Development Cards):** แบ่งเป็น 3 ระดับ (Tier 1, 2, 3) การ์ดแต่ละใบจะมี "ราคา" ที่ต้องจ่าย, "โบนัสอัญมณี" ถาวร, และบางใบมี "คะแนน"
3. **ไพ่ขุนนาง (Nobles):** หากผู้เล่นสะสมโบนัสจากการ์ดได้ครบตามที่ขุนนางต้องการ ขุนนางจะมาหาอัตโนมัติและมอบคะแนนพิเศษให้

**วิธีเล่นในแต่ละเทิร์น:**
เมื่อถึงเทิร์นของคุณ **ต้องเลือกทำ 1 แอคชั่น** จาก 4 ตัวเลือกดังนี้:
1. **หยิบเหรียญอัญมณี 3 สี** สีละ 1 เหรียญ
2. **หยิบเหรียญอัญมณีสีเดียวกัน 2 เหรียญ** (ทำได้ก็ต่อเมื่อกองนั้นมีเหรียญเหลือ 4 เหรียญขึ้นไป)
3. **จองการ์ด 1 ใบ** นำการ์ดบนกระดานขึ้นมือ (เก็บไว้ซื้อทีหลัง) และจะได้รับ **เหรียญทอง 1 เหรียญ** (จองได้สูงสุด 3 ใบ)
4. **ซื้อการ์ด 1 ใบ** จ่ายเหรียญตามราคาที่การ์ดระบุ (หักลบด้วยโบนัสจากการ์ดที่เคยซื้อไปแล้ว) การ์ดที่ซื้อจะให้โบนัสสีถาวรตลอดเกม

**เงื่อนไขการจบเกม:**
เมื่อมีผู้เล่นคนใดคนหนึ่งสะสมคะแนนถึง **15 คะแนน** เกมจะดำเนินต่อไปจนจบวง (เพื่อให้ทุกคนได้เล่นจำนวนรอบเท่ากัน) เมื่อจบรอบ ผู้ที่มีคะแนนมากที่สุดจะเป็นผู้ชนะ *(หากคะแนนเท่ากัน ผู้ที่ซื้อการ์ดน้อยกว่าจะเป็นผู้ชนะ)*

### 🎮 Game Manual (คู่มือการเล่นและ Input)

**การเริ่มต้นเกม:**
1. ทำการ Compile และ Run ไฟล์ `main.cpp`
2. ระบบจะถามจำนวนผู้เล่น `[System]---Please enter the number of player [1-4] : ` ให้พิมพ์ตัวเลขแล้วกด Enter
3. พิมพ์ตั้งชื่อผู้เล่นแต่ละคนตามลำดับ
4. เกมจะทำการ Setup กระดาน สับไพ่ และเริ่มเทิร์นของผู้เล่นคนแรก

**การอ่านหน้าจอกระดาน (Board Display):**
ในแต่ละเทิร์น ระบบจะแสดงผลกระดานส่วนกลาง (จำนวนเหรียญที่เหลือ, ไพ่ที่เปิดอยู่บนโต๊ะพร้อม ID) และสถานะของผู้เล่น (เหรียญที่มี, โบนัสที่มี, คะแนนรวม)

**การสั่งงานในเทิร์น (Input):**
ระบบจะมีเมนูให้เลือกทำ Action โดยให้ผู้เล่นพิมพ์ตัวเลขเพื่อเลือกคำสั่ง ดังนี้:

* **กด `1` : หยิบเหรียญ 3 สี (Take 3 Gems)**
  * ระบบจะให้พิมพ์สีที่ต้องการ เช่น พิมพ์ `White Blue Red` (หรือตัวย่อตามที่โค้ดกำหนด)
* **กด `2` : หยิบเหรียญสีเดียวกัน 2 เหรียญ (Take 2 Gems)**
  * ระบบจะให้พิมพ์สีที่ต้องการ 1 สี เช่น พิมพ์ `Green`
* **กด `3` : ซื้อการ์ด (Buy Card)**
  * ระบบจะให้พิมพ์ **ID ของการ์ด** ที่ปรากฏบนกระดาน (หรือ ID การ์ดที่จองไว้ในมือ) ที่ต้องการซื้อ เช่น พิมพ์ `45`
* **กด `4` : จองการ์ด (Reserve Card)**
  * ระบบจะให้พิมพ์ **ID ของการ์ด** บนกระดานที่ต้องการเก็บขึ้นมือ เช่น พิมพ์ `12`

*(หมายเหตุ: หากผู้เล่นพิมพ์ Input ผิดพลาด หรือทรัพยากรไม่พอซื้อ ระบบจะแจ้งเตือนและให้เลือก Action ใหม่อีกครั้งโดยไม่ข้ามเทิร์น)*
